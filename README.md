# The Onsen Oracle

**Soak in your destiny.** A mystical Japanese hot spring fortune teller for 2026.

Built with vacation vibes in Niseko, Japan on NYE 2024.

---

## What is it?

The Onsen Oracle is a beautiful, shareable fortune teller inspired by Japanese onsen culture. Users "enter the waters" to receive their personalized 2026 fortune, complete with:

- A poetic fortune inspired by hot spring wisdom
- An elemental alignment (Water, Fire, Earth, Wind, Void)
- Personalized guidance for the year ahead
- A lucky number (1-99)

## Features

- Warm, immersive onsen color palette
- Steam particle animations
- Glowing water surface effects
- Golden washi-paper fortune cards
- Native share functionality (or Twitter fallback)
- Mobile and desktop optimized

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + custom animations
- **Typography:** Noto Serif JP + Inter
- **Deployment:** Vercel (free tier)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to soak in your destiny.

The Onsen Oracle won because:
